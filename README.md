# Amazon Clone - Free Frontend Practice Project

![Amazon Clone Screenshot](assets/header1.jpg)

A free and open-source front-end clone of Amazon's homepage built with HTML, CSS, and a touch of JavaScript. This project replicates the layout and design of Amazon's e-commerce platform, including navigation, product sections, and footer. It's perfect for learning and practicing front-end development skills.

## 🚀 Features

- **Responsive Navigation Bar** - Amazon-style logo, location selector, search bar, language selector, sign-in, returns, and cart
- **Category Navigation** - Bottom navigation with links to departments
- **Image Slider** - Hero section with promotional banners (static, but ready for JS enhancement)
- **Product Grids** - Multiple product category boxes with images and "Shop more" links
- **Horizontal Product Sliders** - Scrollable product rows for best sellers and deals
- **Product Cards** - Items with discount badges, prices, and descriptions
- **Footer** - Amazon logo and copyright information

## 🛠 Technologies Used

- HTML5
- CSS3 (linked as `Amazon style.css`)
- JavaScript (basic structure in `script.js`)

## 🚦 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (for modifications)

### Installation
1. Clone the repository or download the files
2. Ensure all images are placed in an `assets/` folder at the root level
3. Create `Amazon style.css` and `script.js` files (or use your own)
4. Open `Amazon.html` in your browser

### Note on Image Paths
The HTML uses absolute paths like `/assets/amazon_logo.png`. If running locally, you may need to:
- Update paths to relative (`assets/amazon_logo.png`) if your server root differs
- Or serve the project from a local server (e.g., Live Server in VS Code) to keep absolute paths working
