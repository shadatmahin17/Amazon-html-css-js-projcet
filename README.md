## рџљ¦ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (for modifications)

### Installation
1. Clone the repository or download the files
2. Ensure all images are placed in an `assets/` folder at the root level
3. Create `Amazon style.css` and `script.js` files (or use your own)
4. Open `Amazon.html` in your browser

### Note on Image Paths
The HTML uses absolute paths like `/assets/amazon_logo.png`. If running locally, you may need to:
- Update paths to relative (`assets/amazon_logo.png`) if your server root differs
- Or serve the project from a local server (e.g., Live Server in VS Code) to keep absolute paths working
